# Unemployment Analysis with Python

📌 Project Description

This project analyzes unemployment rate data in India using Python. The main objective is to study unemployment trends over time, understand the impact of COVID-19, and identify patterns in the data.

🎯 Objectives

* Analyze unemployment rate data (percentage of unemployed people)
* Perform data cleaning and preprocessing
* Explore the dataset using statistical methods
* Visualize trends using graphs
* Study the impact of COVID-19 on unemployment
* Identify seasonal and regional patterns
* Provide insights for economic understanding

🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

📊 Dataset

The dataset contains:

* Region (State)
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Rural/Urban)

🔍 Process

1. Data Cleaning

   * Removed missing values
   * Fixed column names
   * Converted Date column to datetime format

2. Data Analysis

   * Extracted Year and Month
   * Calculated average unemployment rates
   * Compared data before and after COVID-19

3. Data Visualization

   * Line chart for unemployment trend over time
   * Bar chart for year-wise analysis
   * Region-wise unemployment comparison
   * Monthly trend analysis

🦠 COVID-19 Impact

The analysis shows that unemployment increased significantly after COVID-19. The average unemployment rate rose from around 9.39% before 2020 to 15.10% after 2020, indicating the negative impact of the pandemic on employment.

Key Insights

* Unemployment rate increased sharply during COVID-19
* 2020 shows the highest unemployment levels
* Different regions have varying unemployment rates
* Monthly trends indicate seasonal variations
* Data analysis helps in understanding economic conditions


✅ Conclusion

This project demonstrates how Python can be used for data analysis and visualization. It highlights the impact of COVID-19 on unemployment and helps in understanding trends and patterns in the data.
