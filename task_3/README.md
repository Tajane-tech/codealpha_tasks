🚗 Car Price Prediction using Machine Learning

📌 Project Description

This project aims to predict the price of a car based on various features such as present price, kilometers driven, fuel type, transmission, and number of owners. A Linear Regression model is used to understand the relationship between these features and the car’s selling price.

🎯 Objective

* To build a regression model for predicting car prices
* To perform data preprocessing and feature engineering
* To evaluate model performance using appropriate metrics


📂 Dataset Features

The dataset contains the following columns:

* Car_Name
* Year
* Selling_Price (Target Variable)
* Present_Price
* Driven_kms
* Fuel_Type
* Selling_type
* Transmission
* Owner


⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn



🔄 Workflow

1. Load dataset using Pandas
2. Perform data preprocessing
3. Create a new feature (Car Age)
4. Convert categorical data into numerical form
5. Split data into training and testing sets
6. Train the model using Linear Regression
7. Predict car prices
8. Evaluate model using MAE and R² Score
9. Visualize results using a scatter plot



📊 Model Evaluation

* Mean Absolute Error (MAE): Measures average prediction error
* R² Score: Measures accuracy of the model



📈 Visualization

A scatter plot is used to compare actual and predicted car prices.



✅ Conclusion

The model successfully predicts car prices with good accuracy. The results show that machine learning can be effectively used for real-world price prediction problems.
