 Iris Flower Classification (Machine Learning Project)

📌 Task Description

This project focuses on building a Machine Learning model to classify Iris flowers into three species:

* Setosa
* Versicolor
* Virginica

The classification is based on four input features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width



🎯 Objectives

* Use Iris flower measurements as input data
* Train a machine learning model for classification
* Use Scikit-learn library for model building
* Evaluate model performance using test data
* Understand basic classification concepts



📂 Dataset

* File used: **Iris.csv**
* Contains:

  * 150 rows (samples)
  * 5 columns:

    * SepalLengthCm
    * SepalWidthCm
    * PetalLengthCm
    * PetalWidthCm
    * Species


🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn



🧠 Conclusion

The Logistic Regression model successfully classifies Iris flower species with very high accuracy. The dataset is simple and well-structured, making it ideal for understanding basic classification concepts in Machine Learning.
